# kotlin-minesweeper


# 지뢰 찾기

## 기능 목록

### 좌표(지뢰, 땅)
- [x] 좌표는 음수가 될 수 없다.

### 보드
- [x] 보드의 높이, 너비는 음수가 될 수 없다.
- [x] 보드는 높이 * 너비 크기의 좌표를 갖고 있다.

### 지뢰(들)
- [x] 지뢰들은 생성시 인자로 들어온 개수에 해당하는 좌표 정보들을 갖고 있다.
### 지뢰 생성기
- [ ] 지뢰 생성기는 보드 좌표에 범위내의 중복되지 않는 위치에 지뢰들을 생성한다.

### 지뢰 찾기
- [ ] 입력받은 인자에 따라 지뢰와 보드를 생성한다.
